# هيثم المطور — AI Studio Free Tier

نسخة هجينة من أداة توليد الصوت:

- تستخدم `gemini-2.5-flash-preview-tts` مباشرة من المتصفح، مثل النسخة القديمة.
- لا تحتوي على سيرفر مشترك أو مفتاح API مضمّن.
- كل مستخدم يختار **AI Studio Free Tier** أو مفتاحه من قائمة **Secrets**، فتُحتسب الطلبات على حصة حسابه الحالي.
- تضم ثبات الصوت العالي، تحليل المشاعر، التقسيم الذكي، توليد كل الأجزاء معًا أو 5 أجزاء معًا، حفظ الأجزاء الناجحة، استكمال الأجزاء الفاشلة، والدمج النهائي بالترتيب.

## التشغيل داخل Google AI Studio

1. ارفع ملفات المشروع أو استورد ملف ZIP داخل Build في Google AI Studio.
2. افتح **Secrets**.
3. أمام `GEMINI_API_KEY` اختر **AI Studio Free Tier** لاستخدام حصة الحساب الحالي، أو اختر مفتاح Gemini خاصًا بك.
4. اضغط **Reload app**.

عند تبديل حساب Google، افتح/استورد نسختك من التطبيق في ذلك الحساب، ثم اختر **AI Studio Free Tier** من Secrets.

## التشغيل محليًا

أنشئ ملف `.env.local` بنفسك (الملف غير موجود داخل الحزمة):

```env
GEMINI_API_KEY=YOUR_KEY
```

ثم:

```bash
npm install
npm run dev
```

## ملاحظات

- الوضع الافتراضي هو **توليد كل الأجزاء معًا (مثل النسخة القديمة)**، ويمكن تغييره إلى **5 أجزاء معًا** عند الحاجة.
- **ثبات الصوت العالي** مفعّل افتراضيًا.
- ضغط الصوت النهائي غير مفعّل افتراضيًا حتى لا يغيّر الطابع الصوتي، ويمكن تشغيله يدويًا.
- حدود الاستخدام الفعلية يحددها Google حسب الحساب والمنطقة والنموذج، ولا يمكن للتطبيق تجاوز الحصة بعد استنفادها.
